# Car Sales Analytics Using Python, SQL, and Power BI

## Project Overview

This project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI. The objective was to analyze car sales data, identify sales trends, evaluate brand and regional performance, and generate actionable business insights.

The raw dataset was cleaned and transformed using Pandas, then loaded into MySQL for analysis. SQL queries were used to answer key business questions related to revenue, sales performance, fuel type preferences, and monthly trends. Finally, an interactive Power BI dashboard was created to visualize the findings and support data-driven decision-making.

## Tools Used

* Python (Pandas)
* MySQL
* Power BI

## Project Workflow

1. Data Cleaning and Transformation using Pandas
2. Loading Cleaned Data into MySQL
3. Performing SQL Analysis
4. Creating Interactive Power BI Dashboards
5. Generating Business Insights and Recommendations

## Key Analyses

* Brand-wise Revenue Analysis
* Region-wise Sales Performance
* Fuel Type Distribution Analysis
* Monthly Sales Trends
* Top Performing Brands
* Top Performing Car Models
* Revenue Contribution Analysis

## Key Insights

* North region generated the highest revenue.
* Petrol cars were the most preferred vehicle type among customers.
* Top-performing brands contributed significantly to overall sales.
* Monthly sales analysis revealed variations in customer demand across different periods.

## Business Recommendations

* Increase inventory for top-selling brands and models.
* Strengthen marketing efforts in high-performing regions.
* Improve sales strategies in lower-performing regions.
* Prioritize stocking and promoting petrol vehicles to align with customer preferences.
* Use monthly sales trends for better demand forecasting and inventory planning.

## Dashboard Features

* Revenue KPIs
